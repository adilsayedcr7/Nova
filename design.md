# NovaMLM Software — Design System Specification

## Core Directives:
1. **Exclusive Font**: **Figtree** ONLY (`font-family: 'Figtree', sans-serif`).
2. **Background Color**: `#08070B` (Sleek Dark Shade of Black) for page background. `#121017` for card surfaces.
3. **Official Logo Lockup**:
   - Geometric block `N` icon in Vibrant Blue `#0090FF` & `#1164F0`.
   - Wordmark: `Nova` (white bold) `MLM` (gradient cyan-blue `#5EEBFC` -> `#0090FF`) with `Software` (white subtext) underneath.

## Palette
```css
:root {
  --navy-dark:    #08070B;   /* Sleek Dark Black */
  --navy-card:    #121017;   /* Raised Dark Surface */
  --blue-primary: #1164F0;   /* Primary Electric Blue */
  --blue-vibrant: #0090FF;   /* Vibrant Cyan Blue (CTAs) */
  --cyan-light:   #5EEBFC;   /* Light Cyan Highlight */
  --white:        #FFFFFF;   /* Primary Text */
  --smoke-blue:   #A19CBB;   /* Secondary Copy */
}
```
