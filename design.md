# Nova — Design System Specification

## Core Directives:
1. **Exclusive Font**: **Figtree** ONLY (`font-family: 'Figtree', sans-serif`).
2. **Background Color**: `#08070B` (Sleek Dark Shade of Black) for page background. `#121017` for card surfaces.
3. **Official Logo Lockup**:
   - Geometric block `N` icon in Confident Royal Blue `#2563EB` & `#1D4ED8`.
   - Wordmark: Clean, bold **`Nova`** (White, font-weight 800).

## Palette
```css
:root {
  --navy-dark:    #08070B;   /* Sleek Dark Black */
  --navy-card:    #121017;   /* Raised Dark Surface */
  --blue-primary: #1D4ED8;   /* Deep Corporate Blue */
  --blue-vibrant: #2563EB;   /* Royal Blue (CTAs) */
  --cyan-light:   #60A5FA;   /* Soft Sky Blue */
  --white:        #FFFFFF;   /* Primary Text */
  --smoke-blue:   #B8B3D3;   /* Secondary Copy */
}
```
